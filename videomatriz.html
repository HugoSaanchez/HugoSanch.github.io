Implementación de un Sistema de Matriz de Vídeo

Resumen

Este proyecto consiste en desarrollar una aplicación que gestiona una matriz de vídeo, permitiendo la distribución de señales de vídeo desde varios transmisores a múltiples receptores. Los receptores pueden conectarse a un solo transmisor a la vez, mientras que los transmisores pueden enviar señal a múltiples receptores simultáneamente. La aplicación permite configurar y gestionar esta matriz, facilitando la organización de transmisores y receptores, y guardando configuraciones en perfiles para una fácil selección.

Herramientas y Tecnologías

La aplicación se desarrollará en .NET 8 (C#) utilizando Blazor Server y Visual Studio. Se puede usar MySQL para la base de datos y se permiten paquetes NuGet según sea necesario.
