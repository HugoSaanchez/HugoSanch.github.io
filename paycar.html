PayCar

PayCar es una aplicación creada para resolver el problema de la distribución equitativa de los gastos de gasolina durante viajes en grupo. Muchas veces, el conductor asume la mayor parte de los costos, lo que genera desacuerdos entre amigos. Con PayCar, los gastos de gasolina se registran y dividen automáticamente entre los pasajeros, proporcionando una solución práctica, justa y transparente para todos los involucrados.

Funcionalidades principales:

  • Registro de gastos: El conductor puede registrar los gastos de gasolina en la aplicación, lo que facilita el control y monitoreo de los costos durante el viaje.
  • División automática de costos: La app calcula automáticamente el costo total de la gasolina y lo divide equitativamente entre los pasajeros, asegurando que todos paguen un porcentaje justo.
  • Chat integrado: Permite a los viajeros comunicarse entre sí a través de un sistema de mensajería incorporado en la aplicación, facilitando la organización del viaje.
  • Alquiler de vehículos: PayCar ofrece la posibilidad de alquilar vehículos, permitiendo a los usuarios gestionar tanto los gastos de gasolina como la contratación de coches para sus viajes.
  • Transparencia en los pagos: Tanto el conductor como los pasajeros pueden visualizar de manera clara quién ha pagado y cuánto, promoviendo la equidad en la distribución de los costos.

Herramientas y Tecnologías:

Backend:

  • Desarrollado en Java utilizando Spring como framework principal para la lógica del servidor.
  • Base de datos gestionada con MySQL.
  • Control de versiones implementado con GitHub.
  • Despliegue realizado en Render con base de datos en la nube utilizando Railway.

Frontend:

  • Aplicación móvil creada con Flutter utilizando Dart como lenguaje de programación.
  • Integración con el backend a través de una API desarrollada en Eclipse.
  • Pruebas de API realizadas con Postman.
